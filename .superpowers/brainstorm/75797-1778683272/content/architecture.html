Interoperability via site identifier — Every record in every layer carries a persistent site ID (e.g. NBD-MARA-001). The ID is pre-loaded in the KoboCollect form, embedded in admin records, GEE pipeline outputs, lab QA records, FGD session records, and all portal API responses. Any external dataset can join to platform data using this key alone — no custom field mapping required.   ·   Stack: PostgreSQL + PostGIS · FastAPI · Next.js · ECharts · Leaflet.js · Docker · GitHub Actions
